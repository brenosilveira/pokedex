# Pokedex

## Pokedex com 300 Pokemons usando JavaScript puro (Vanilla JS)

Através de uma **API** já pronta que se encontra [nesse site](https://pokeapi.co/), foi feito um request pelo **ID** de cada pokemon. Após isso, cada pokemon foi colocado dentro de um card contendo numero (id), nome e tipo.
As imagens para cada card foram pegas de um [outro site](https://pokeres.bastionbot.org/) através de seus respectivos **ID's** também.
